# CodeAlpha_Hangman 🎮

A desktop Hangman game built in Python using Tkinter, created for the CodeAlpha Python Programming Internship.

## About
This is a classic word-guessing game with a graphical interface (not just text in a terminal). The player guesses letters to reveal a hidden word before running out of chances. Each wrong guess draws another part of the hangman figure on screen.

## Features
- Visual hangman drawing that builds stage by stage on wrong guesses
- 4 word categories: Programming, Animals, Countries, Movies
- 3 difficulty levels: Easy, Medium, Hard (changes word length and allowed guesses)
- Clickable on-screen keyboard (or just type on your real keyboard)
- Win/loss score tracker that updates across rounds
- "New Game" button to restart anytime

## How to Run
1. Make sure Python 3 is installed
2. Download `hangman_gui.py`
3. Open a terminal in that folder and run:
   
  ** _python3 hangman_gui.py_***
  
4. A game window will open — start guessing!

## Concepts Used
Object-oriented programming (classes), Tkinter GUI development, Canvas drawing, event handling, dictionaries, random module, functions, state management
